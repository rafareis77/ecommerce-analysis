# Análise de E-commerce

Análise estatística do comportamento de usuários em uma plataforma de e-commerce.

## Estrutura
- `data/` — Dataset com 500 usuários (visitas, tempo, itens, compras)
- `analysis/` — Script Python com análise estatística e visualizações
- `reports/` — Relatório final com insights e recomendações

## Dependências
pip install numpy pandas seaborn matplotlib

## Como executar
python analysis/AnaliseEcommerce.py
